# OpenCode Modes for Pi

This extension makes Pi follow the OpenCode-style plan/build permission config
generated by the Nix module at `~/.pi/agent/opencode-permissions.json`.

Commands:

- `/plan` switches to read-only plan mode.
- `/build` switches to build mode.
- `/mode` shows or changes the current mode.

Flags:

- `--plan` starts a session in plan mode.

The permission source of truth is `configs/opencode/permissions/*.json`.
