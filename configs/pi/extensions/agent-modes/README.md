# Agent Modes for Pi

This extension makes Pi follow the shared agent plan/build permission config
generated by the Nix module at `~/.pi/agent/agent-permissions.json`.

Commands:

- `/plan` switches immediately to read-only plan mode.
- `/build` switches immediately to build mode.
- `/mode` shows or changes the current mode.

Keyboard shortcuts:

- `Tab` selects plan/build mode for the next submitted message. It updates the
  status line immediately, but the active permissions do not change until you
  submit the message.

Flags:

- `--plan` starts a session in plan mode.

The permission source of truth is `configs/agents/permissions/*.json`.
